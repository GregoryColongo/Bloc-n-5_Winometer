Wine O Meter API : find out the quality of wine

How doest it work ?

Wine O Meter app allows you to predict the quality score of your wine based on physicochemical inputs. The score goes from 0 to 10.

Description of endpoints

http://127.0.0.1:5000/predict

This endpoint will be used for the quality wine prediction

HTTP methods

This endpoint accepts POST method with JSON input data and it should return the predictions

Required input and expected output

Please follow these steps (either step 1 for CURL or step 2 for PYTHON):



  1. Call the endpoint with CURL

    2. Curl:

    $ curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' http://127.0.0.1:5000/predict


  2. Call the endpoint with PYTHON

    3. Python:

    import requests

    response = requests.post("http://127.0.0.1:5000/predict", json={ "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]] })

    print(response.json())


  3. Fill in the 11 values as float type inside a JSON format as below

    4. Input example:

    { "input": [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]] }

    The response should be a JSON with one key prediction corresponding to the prediction.

    Response example: { "prediction":[6]}



    !! You can also put multiple inputs in the same request !!

    { "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]] }

    Response example: { "prediction":[6,6] }

  4. Good tasting and cheers !
